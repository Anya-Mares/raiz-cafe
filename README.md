# Raíz Café

Proyecto frontend desarrollado con Next.js y React.

## Tecnologías
- Next.js
- React
- CSS Modules
- Vercel

## Descripción
Sitio web conceptual para una marca de café artesanal.
Diseño minimalista enfocado en experiencia visual, responsividad
y estructura de componentes reutilizables.

## Créditos de imágenes
Imágenes utilizadas con fines demostrativos.
Provenientes de bancos de imágenes gratuitos como Unsplash / Pexels.
Derechos pertenecen a sus respectivos autores.
